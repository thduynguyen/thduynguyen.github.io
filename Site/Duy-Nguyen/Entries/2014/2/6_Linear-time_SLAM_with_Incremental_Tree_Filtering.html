Linear-time SLAM with Incremental Tree Filtering

 

For my thesis work, I explored object-centric based approaches for fast object estimation and robot localization. My exploration led to linear-time approximation algorithms for information filtering-based SLAM, which is traditionally cubic with respect to the number of features of the object. More details in my IROS paper.
